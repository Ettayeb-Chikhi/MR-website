Vue js

Vue.js est disponible pour le public en 2014. Son créateur, Evan You, l'a créé car il ne trouvait pas l'aisance de travail dans les solutions proposées par les autres frameworks JavaScript. En effet, deux ans avant la sortie de Vue.js, Evan You avait décidé d'apprendre le JavaScript. Et, tout simplement, il ne trouvait pas un framework qui lui convenait. Ni une ni deux, il se créa une solution perso qui regroupait ce qu'il aimait dans les différents frameworks JavaScript qu'il apprenait ! Vue.js n'était alors utilisé que pour ses projets, c'était SON outil de travail. Conscient qu'il avait là une solution légère et malléable, exactement l'inverse de ce qu'il reprochait aux autres frameworks, il finira par rendre disponible sa solution au plus grand nombre.

Vue.js est un framework JavaScript open-source qui permet de construire des interfaces utilisateur interactives pour les applications web. Il utilise une architecture basée sur des composants pour faciliter la réutilisation et la maintenance du code.

Vue.js est principalement utilisé pour le développement de single-page applications (SPA), où une seule page web est chargée et modifiée dynamiquement à mesure que l'utilisateur interagit avec l'application. Il offre également une bibliothèque de directives, qui sont des attributs personnalisés qui peuvent être utilisés pour ajouter des fonctionnalités à un élément HTML, ainsi qu'un système de gestion d'état basé sur une architecture flux.
